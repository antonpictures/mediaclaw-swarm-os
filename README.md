## Installation & Quick Start

Runtime: **Node ≥22**.

1. Clone the repository:
   ```bash
   git clone https://github.com/openclaw/openclaw.git media_claw
   cd media_claw
   ```

2. Run the onboarding wizard:
   ```bash
   npm install -g openclaw@latest
   openclaw onboard --install-daemon
   ```

3. Spin up the Autonomous Trust Machine:
   ```bash
   cd newspaper
   npm install
   npm run build
   npm run start
   ```
   *Note: Ensure your Python specialized agents (like `alice_youtube_agent.py`) are provisioned within the skills directory for autonomous generation.*

Once installed, simply drop a YouTube link into your dashboard, and watch your local Sovereign Node construct the news, cryptographically secure the ledger, and connect humanity to a bright future of information.
